Modal

Data Attributes

Below are some notes about some of the available data attributes.

data-pat-modal-position
Optional, the default being center middle.

In the form <horizontal position> <vertical position>, where "<horizontal position>" is one of center, left, right, and where "<vertical position>" is one of middle, top, bottom. Ex: "center middle"

If you wanted to position a modal on the left side of the page, but didn't want it positioned 0px from the left, you'd want to use the data-pat-modal-margin option.
data-pat-modal-margin
This is ultimately an integer value. Defined through a data attribute it must be an integer value -- but defined through javascript, it can be an integer value or a function that takes no arguments and returns an integer value.

The integer value represents a unit amount in pixels for a margin on all 4 sides of the modal dialog.

Related Items

The Related Items widget is intended to make it easy for a user editing content to locate content that is related to the current piece of content.

To this end, the widget has two "modes" -- the first is when a user is entering text that they'd like to match against content in their website, and the second is when a user is point-and-click browsing through the hierarchy of their website.

The widget makes use of the Select2 by customizing it's interface for this specific purpose.

A request to a service that returns a JSON file is used to gather information by the Select2 widget. The service understands a few parameters that get past to it in order to return the relevant set of data based on the query and location within the query results the desired data resides.

Picture

Responsive images pattern

This pattern load the last image that fulfill the media-query criteria. When multiple pictures are displayed it loads the images within the visible viewport first. Will update images on resize event.

Data Attributes

Below are some notes about some of the available data attributes.

data-pat-picture-alt
Alternative text used on img tag
data-pat-picture-klass-loading
CSS class added to element while loading the image
data-pat-picture-klass-error
CSS class added to element if error occured while loading the image
data-pat-picture-attribute-src
Which attribute to fetch image source information from, defaults to data-src
data-pat-picture-attribute-media
Which attribute to fetch image mediaquery information from, defaults to data-media
      <div class="pat-picture" data-pat-picture-alt="Alternative text">
       <div data-src="http://placehold.it/480x320"></div>
       <div data-src="http://placehold.it/640x427" data-media="(min-width: 480px)"></div>
       <div data-src="http://placehold.it/800x533" data-media="(min-width: 640px)"></div>
       <div data-src="http://placehold.it/960x640" data-media="(min-width: 800px)"></div>
       <div data-src="http://placehold.it/1200x800" data-media="(min-width: 960px)"></div>
       <!--[if (lt IE 9) & (!IEMobile)]>
         <div data-src="http://placehold.it/800x533"></div>
       <![endif]-->
       <noscript>
         <img src="http://placehold.it/800x533" alt="Alternative text" />
       </noscript>
      </div>
