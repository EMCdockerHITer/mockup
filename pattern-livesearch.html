Live Search

Default

<div class="pat-livesearch"
  data-pat-livesearch="ajaxvocabulary: /search.json;">
  <input type="text" class="pat-livesearch-input"
    placeholder="Search" />
  <div class="pat-livesearch-container">
    <div class="pat-livesearch-results">
    </div>
  </div>
</div>

Data Attributes

Attribute Type Default Description
ajaxvocabulary String The url that will be of the server side page that will be queried.
delay Integer 200 Delay in ms after user types before searching.
highlight String 'pat-livesearch-highlight' CSS class that will be used to highlight items on keyboard navigation.
minimumInputLength Integer 3 Number of characters user must type before search begins.
input Selector '.pat-livesearch-input' Selector for element that will be used for the livesearch input.
toggleTarget Selector '.pat-livesearch-container' Selector for element that will be shown when results are found.
toggleClass String 'show' CSS class that will be added to target toggle element when results are found.
positionToggleWithInput Boolean true Whether or not the toggle element should be positioned relative to the input element. If set to false setPosition() will not be called.
setPosition Function 
setPosition: function() {
  var self = this;
  if (self.options.positionToggleWithInput) {
    var top = self.$input.position().top + self.$input.outerHeight();
    self.$toggle.css({'top': top+'px'});
  }
}
 Function which will be used to set the position of the toggle element.
resultsTarget Selector '.pat-livesearch-results' Selector for element where results from the server will be inserted.
resultContainerTemplate String '<ul></ul>' Template for the element that will contain results. If resultContainerTemplateSelector is set, this will not be used.
resultContainerTemplateSelector Selector Selector for an element in the DOM to use in place of resultContainerTemplate.
resultSelector Selector '.pat-livesearch-result' Will be used to select result items from within the result container.
resultTemplate String 
'<li class="pat-livesearch-result pat-livesearch-type-<%= Type %>">' +
  '<a class="pat-livesearch-result-title" href="<%= getURL %>">' +
    '<%= Title %>' +
  '</a>' +
  '<p class="pat-livesearch-result-desc"><%= Description %></p>' +
'</li>'
 Template for a single result.
resultTemplateSelector String Selector for element to be used in place of resultTemplate.
helpTemplate String '<div class="pat-livesearch-help"><%= help %></div>' Template for the element that will contain help messages, such as 'No results found.' or 'Type 3 more characters to search.'
helpTemplateSelector Selector Element in the DOM to be used instead of helpTemplate.
typeMoreTemplate String 'Type <%= more %> more characters to search.' Template for message which is displayed if the current input length is less than minimumInputLength.
typeMoreTemplateSelector Selector Element in the DOM to be used instead of typeMoreTemplate.
typeMoreTemplateSelector Selector Element in the DOM to be used instead of typeMoreTemplate.
noResultsTemplate String 'No results found.' Template for message that is displayed after a search when no results were found.
noResultsTemplateSelector Selector Element in the DOM to be used instead of noResultsTemplate.
searchingTemplate String 'Searching...' Template for message which will be displayed after a search has started and before it has ended.
searchingTemplateSelector Selector Element in the DOM to be used instead of searchingTemplate.
