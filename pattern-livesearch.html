Data Attributes

The livesearch pattern is built on top of the select2 pattern which means that many of the same options and attributes will work here as well. For more information on select2 refer to the Select2 pattern docs and the overall select2 library docs.

Attribute Default Description
ajaxvocabulary The url of the server side page that will be queried.
minimumInputLength 3 Number of characters user must type before search begins.
